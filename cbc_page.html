A valid library card, or photo ID of a cardholder, from one of the libraries listed below will be honored at any of the libraries. Scroll down for information about each library. You may want to call to verify library hours and policies before visiting. If summer hours differ from the regular schedule they are noted.

    • Amherst Town Library
    • Bedford Public Library
    • Danforth Library (New England College)
    • Derry Public Library
    • Goffstown Public Library
    • Hooksett Public Library
    • Kelley Library (Salem)
    • Manchester City Library
    • Manchester West Branch Library
    • Merrimack Public Library
    • Nesmith Library (Windham)
    • Rodgers Memorial Library (Hudson)
    • Wadleigh Memorial Library (Milford)

    A valid card is one that has not expired, has no charges for lost materials, and no excessive fines as defined by the lending library.

    1. Fines may be paid at any library, (except NEC) but lost items must be paid to the owning library.
    2. Policies of the lending library prevail.
    3. Interlibrary Loan and Museum Passes are only available at your local library.
    4. Materials may be returned to any participating library.
    5. No renewals are allowed on materials which have holds on them.
    If you are looking for something specific, please call the library to verify lending policies and hours.
